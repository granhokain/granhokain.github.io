Sobre

Eu sou estudante de ciência da computação na Universidade Tiradentes. Minhas principais áreas de interesse são Desenvolvimento Web, Desenvolvimento Android, Código Aberto, Estrutura de Dados e Algoritmos. Eu também tenho explorado ultimamente o framework open source para plataforma Java, o Spring.

Escrevi um artigo em 2017 publicado no caderno de graduação da instituição sobre o desenvolvimento de um aplicativo android, o Enade Simulado, para auxiliar nos estudos dos participantes do ENADE - Exame Nacional de Desenpenho dos Estudantes.

Eu sou apaixonado por código aberto. Eu tenho uma boa experiência com a tecnologia GIT, gosto de cada aspecto e adoro criar sites e aplicativos para dispositivos móveis do início ao fim. Eu desenvolvi alguns aplicativos web front-end, sendo as aplicações web responsivas. Também desenvolvi pequenos aplicativos para Android que estão disponíveis na minha conta do GitHub para fins de aprendizado.

Fiz o meu estágio em algumas empresas ampliando meus conhecimentos, adquirindo e melhorando minhas habilidades. Fiz estágio na Avonale, trabalhando com tecnologia C# .NET, Knockout, Windows e Web Forms. Também estagiei na Secretaria da Saúde, trabalhando como desenvolvedor PHP atualizando módulos no Scriptcase usando o banco de dados PostgreSQL.

Projetos

Exemplos de alguns projetos desenvolvidos

Enade Simulado

 É um app android que auxilia na preparação para o exame do Enade. O app possui um sistema semelhante à de um quiz, onde o aluno responde a questões de múltipla escolha e ao final do questionário visualiza seus acertos e erros no simulado.

Archery Game

É um jogo 2D em javascript de tiro ao alvo voltado para deficientes visuais, possuindo recursos como efeitos sonoros, onde sempre que a mira passa no alvo, é emitido um som onde o jogador mesmo sem ver, possa saber o momento de disparar a flecha.

RVM Website

É um site de apresentação criado com HTML e CSS utilizando de algumas tecnologias e técnicas como parallax, com descrição de alguns projetos como portifólio.
